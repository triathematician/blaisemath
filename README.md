# blaisemath
**blaisemath** is a collection of Java libraries for visualization and mathematics. The source code, javadocs, and jar files are available in the [central maven repository](https://central.sonatype.com/search?q=g:com.googlecode.blaisemath).

Project documentation is available at http://triathematician.github.io/blaisemath/ (TBD)
