This package contains special-purpose utility methods for computing metrics associated with graphs, such as centrality metrics.

Most metrics are subclassed from NodeMetric, which provides a method for getting the value of a metric at a single node, and a method for getting the values of a metric for all nodes. If metrics do not depend upon parameters, generally their class will have a getInstance() method returning a singleton instance of the metric.

The subset subpackage provides classes for metrics that are defined for more than one node, as well as special implementations AdditiveSubsetMetric and ContractiveSubsetMetric that are derived from single-vertex metrics.
