DenseMatrix
public class RowMajorMatrix
extends ContiguousMatrix

A dense contiguous row-major matrix of double values. This storage format matches the 'C' language matrix format. The elements are store contiguously in a single java array. This matrix is efficient for numerical computing, especially row-wise computations. There is more information available about the internal storage format.
