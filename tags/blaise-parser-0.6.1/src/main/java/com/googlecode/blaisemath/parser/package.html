This package contains functionality to recognize "tokens" of various type in a String input, and to convert them into a SemanticNode, which has functionality similar to a method.

Errors in input are handled by the ParseException class.

Extensibility is handled primarily by the Grammar interface, which is used by the tokenizer to determine the types of tokens corresponding to different symbols. Currently supported grammars are the BooleanGrammar and the RealGrammar.
