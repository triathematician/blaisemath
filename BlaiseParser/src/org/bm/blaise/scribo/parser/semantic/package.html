This package consists of classes that are useful for generating semantic trees. Generally the implementation is abstract, so that further classes are required to actually parse a semantic tree. The main entry point is SemanticTreeBuilder, which converts a "token tree", in which the structure of the tree is known but not the meaning, into a "semantic tree", in which the nodes are all mapped to some sort of "meaning". Classes that build trees of different types should implement SemanticTreeBuilder in order to implement the semantic meaning.

The semantic tree is a combination of SemanticNode's, which may have dynamic type parameters.

Errors in evaluation are handled by the SemanticTreeEvaluationException.
